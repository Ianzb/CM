Java版导入整合包教程
1：使用 Hmcl启动器>版本列表>导入整合包 或 Pcl2启动器>下载>整合包>导入已有整合包 即可导入整合包
注：Pcl2启动器导入整合包因为Bug可能报错，建议使用Hmcl启动器导入
如果没有识别出整合包请按照下面的教程
3：用压缩软件打开整合包，寻找有没有启动器程序，如果有启动器，解压整合包后直接运行即可
4：如果没有启动器，也将整合包解压
5：确认你是否打开版本隔离， Pcl2启动器>设置>右侧寻找版本隔离选项>改成隔离所有版本 或 Hmcl启动器>设置>右侧寻找版本隔离选项>改成各版本分离
6：进入 Pcl2启动器>左下角版本选择>鼠标悬停到左上角文件夹列表中你想导入的目录>点击出现的齿轮>点击打开>在打开的文件夹中进入versions目录 或
Hmcl启动器>版本列表>点击右侧任意版本右侧3个点按钮>游戏文件夹>进入打开的文件夹的上级目录
注：如果你此时看见的文件夹名字是versions中有你之前下载的其他版本游戏，就说明正确，下文将此文件夹称为“文件夹1”
7：在解压后的整合包中寻找一个里面有mods等文件夹的文件夹（下文称此文件夹为“文件夹2”），查看“文件夹2”是否包括与整合包名字或游戏版本名字相同的.jar和.json文件
8：如果没有，进入“文件夹2”中的versions文件夹（通常里面还有一个文件夹），你最终会看见上一条中的.jar和.json文件，把拥有这两个文件的文件夹里面的文件全部移动到“文件夹2”
9：在之前打开的“文件夹1”，新建一个和上两条中的.jar和.json文件名称一样的文件夹（不包括后缀名），把“文件夹2”中的全部文件复制到新建的文件夹中
10：打开启动器查看版本就能看到你的整合包
