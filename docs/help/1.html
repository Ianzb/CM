我的世界教程

Java版使用核心开服教程
1.首先，我们需要一个核心，个人推荐paper和spigot
spigot链接：https://getbukkit.org/download/spigot
paper链接：https://papermc.io/downloads
2.我们需要建一个文件夹，把核心放进去（命名使用英文）
3.我们新建一个txt（记事本），然后输入：
java -Xmx1G -Xms1G -jar 核心名称.jar
注：必须要一样，不能缺。
安装好适用版本的Java。
4.我们将记事本保存退出并将后缀名改成.bat，然后运行！
后面有说明教程。
5.运行之后，我们的文件夹会出现一个叫eula.txt的文件，我们打开后将底下的flase改成true（此方法在所有核心均为适用），然后保存并关闭！
6.找到第4项的.bat文件，然后运行！
7.运行完之后整个服务器就全了，以后就用.bat文件启动服务器核心！
8.将在.bat窗口输入stop，以后也要这样关闭服务器！
9.我们需要网络穿透，目前有:
路由侠，樱花，花生壳等
路由侠：http://www.luyouxia.com下载并安装，登录账户，没有注册的注册后登录。
10.打开路由侠（其他内网穿透软件可能有所不同），点添加，然后选第一个，选择自己觉得好的线路，在本地IP输入：127.0.0.1
端口输入:25565（如果自己调过就调一下）。
11.点击开启。
