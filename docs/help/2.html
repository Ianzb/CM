Java版局域网转公网开服教程
1：进入你要联机的游戏。
2：在暂停界面点击在局域网中开放。
3：聊天区会出现一个端口，记住这串数字。
4：打开路由侠或其他内网穿透软件。
5：点击添加-选择基础端口映射并点击确定-左上角随便选择一个服务线路，右侧映射端口写上刚才的数字，其他设置不要改，点击添加即可。
6：回到主页，你会看见已经创建成功，右键你的服务器的外网访问地址并复制发给好友即可。
注：如果你退出了游戏，服务器也会关闭。
如果你想再次开服，发现端口号改变，不需要重新重复第五步，只需要选择之前的服务器并点击修改，在其中修改端口号即可。
除了房主只有正版玩家能进服，加入模组 简单联机1.7.2-1.12.2Forge 或 自定义局域网联机1.12.2-最新版Forge 1.14.4-最新版Fabric 在游戏中局域网联机界面会有允许离线相关设置。
